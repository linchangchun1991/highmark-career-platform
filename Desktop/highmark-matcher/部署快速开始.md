# 🚀 5分钟快速部署指南（超简单版）

如果你是第一次部署，按照这个顺序来，每一步都有详细说明。

## 📌 第一步：配置数据库（Supabase）- 10分钟

### 1. 注册账号
- 访问：https://supabase.com
- 用 GitHub 登录（最简单）

### 2. 创建项目
- 点击 "New Project"
- 项目名：`highmark-matcher`
- 密码：设置一个强密码（记住！）
- 地区：选 "Singapore"（离中国最近）
- 计划：选 "Free"
- 等待 2-3 分钟

### 3. 获取密钥
- 左侧菜单 → Settings → API
- 复制这两个值：
  - **Project URL**（类似 `https://xxxxx.supabase.co`）
  - **anon public key**（一长串，以 `eyJ` 开头）

### 4. 创建数据表
- 左侧菜单 → SQL Editor → New query
- 复制粘贴这段代码：

```sql
CREATE TABLE jobs (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  raw_text TEXT NOT NULL,
  company TEXT NOT NULL,
  roles TEXT NOT NULL,
  location TEXT NOT NULL,
  link TEXT NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE INDEX idx_jobs_created_at ON jobs(created_at DESC);

ALTER TABLE jobs ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Allow public read access" ON jobs FOR SELECT USING (true);
CREATE POLICY "Allow public insert" ON jobs FOR INSERT WITH CHECK (true);
```

- 点击 "Run" 执行

✅ **第一步完成！** 保存好你的 Supabase URL 和 Key

## 📌 第二步：配置 AI（DeepSeek）- 5分钟

### 1. 注册账号
- 访问：https://platform.deepseek.com
- 用手机号或邮箱注册

### 2. 充值
- 进入 "余额" 页面
- 最低充值 5 元（可以用很久）

### 3. 创建 API Key
- 找到 "API Keys" 或 "密钥管理"
- 点击 "创建新密钥"
- **立即复制**（只显示一次！）

✅ **第二步完成！** 保存好你的 DeepSeek API Key

## 📌 第三步：选择部署方式

### 🟢 方式 A：Vercel（最简单，推荐新手）

**适合**：想快速上线，不介意国内访问稍慢

**步骤**：
1. 访问：https://vercel.com
2. 用 GitHub 登录
3. 点击 "Add New Project"
4. 选择你的代码仓库（如果没有，先在 GitHub 创建）
5. **重要**：在 "Environment Variables" 添加：
   - `NEXT_PUBLIC_SUPABASE_URL` = 你的 Supabase URL
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY` = 你的 Supabase Key
   - `DEEPSEEK_API_KEY` = 你的 DeepSeek Key
6. 点击 "Deploy"
7. 等待 2 分钟，完成！

**得到网址**：`https://你的项目名.vercel.app`

---

### 🟡 方式 B：阿里云（国内访问快）

**适合**：需要国内快速访问，愿意花 50-100 元/年

**详细步骤**：看 `部署指南-国内版.md` 的 "第五步"

**简单步骤**：
1. 在阿里云购买服务器（2核2G，约 50-80元/年）
2. 用 SSH 连接服务器
3. 运行快速部署脚本：`bash 快速部署脚本.sh`
4. 按照提示完成配置

## ❓ 遇到问题？

### 问题 1：部署后显示错误
**解决**：检查环境变量是否正确配置

### 问题 2：数据库连接失败
**解决**：
- 检查 Supabase URL 和 Key 是否正确
- 确认数据表已创建（在 SQL Editor 中检查）

### 问题 3：AI 匹配失败
**解决**：
- 检查 DeepSeek API Key 是否正确
- 确认账户有余额

### 问题 4：Vercel 国内访问慢
**解决**：
- 使用阿里云部署（方式 B）
- 或配置 CDN 加速
