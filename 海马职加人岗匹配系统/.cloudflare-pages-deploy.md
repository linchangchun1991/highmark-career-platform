# Cloudflare Pages 部署配置

## 正确配置步骤

### 方法一：使用 Cloudflare Pages（推荐，不需要 wrangler）

1. **访问 Cloudflare Pages**
   - https://dash.cloudflare.com/
   - 选择 "Workers & Pages"
   - 点击 "Create application" → "Pages" → "Connect to Git"

2. **连接 GitHub 仓库**
   - 选择：`linchangchun1991/highmark-career-platform`

3. **配置构建设置**
   - Framework preset: **None** 或 **Other**
   - Build command: (留空)
   - Build output directory: `海马职加人岗匹配系统`

4. **点击 "Save and Deploy"**

### 方法二：如果必须使用 wrangler

需要将静态文件部署为 Workers，配置如下：

```toml
name = "product-price-query"
compatibility_date = "2024-01-01"

[site]
bucket = "./海马职加人岗匹配系统"
```

但建议使用方法一，更简单。
