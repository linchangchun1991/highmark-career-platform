# 产品价格查询系统

海马职加事业部产品价格查询系统，支持多种查询方式和智能优惠计算。

## 功能特性

- 🎨 **现代化界面**：硅谷科技大厂风格的蓝紫渐变设计
- 🔍 **多种查询方式**：支持产品名、交付方、行业、地点、公司名等多种查询
- 💰 **智能优惠计算**：自动计算最大优惠，优惠不叠加（一口价优先）
- 📊 **清晰展示**：显示标准售价、事业部最大优惠、最大优惠后价格
- 📱 **响应式设计**：完美适配桌面和移动设备
- ⚡ **性能优化**：搜索结果缓存，快速响应

## 文件说明

- `product-price-query.html` - 主应用文件（单文件HTML应用）
- `products_with_policies.json` - 产品数据文件（包含产品信息和优惠政策）

## 使用方法

### 本地使用

1. 确保 `products_with_policies.json` 文件与 HTML 文件在同一目录
2. 使用现代浏览器打开 `product-price-query.html`
3. 如果遇到跨域问题，可以使用本地服务器：
   ```bash
   # 使用 Python
   python3 -m http.server 8000
   
   # 或使用 Node.js
   npx serve .
   ```
4. 在浏览器中访问 `http://localhost:8000/product-price-query.html`

### 部署到 Vercel

1. **准备文件**
   - 确保 `product-price-query.html` 和 `products_with_policies.json` 在项目根目录

2. **创建 Vercel 配置**（可选）
   创建 `vercel.json`：
   ```json
   {
     "routes": [
       {
         "src": "/product-price-query.html",
         "dest": "/product-price-query.html"
       },
       {
         "src": "/products_with_policies.json",
         "dest": "/products_with_policies.json"
       }
     ]
   }
   ```

3. **部署步骤**
   - 将项目推送到 GitHub
   - 在 Vercel 中导入项目
   - Vercel 会自动检测并部署
   - 访问部署后的 URL

4. **更新数据**
   - 更新 `products_with_policies.json` 文件
   - 推送到 GitHub，Vercel 会自动重新部署

## 优惠计算规则

系统根据产品政策自动计算优惠，优先级如下：

1. **一口价**（最高优先级，不叠加）
   - 如果优惠中包含"一口价"，直接使用一口价金额

2. **百分比折扣**
   - 如"9折"、"98折"、"95折"、"92折"
   - 计算公式：标准售价 × 折扣率

3. **条件折扣**
   - 如"4w以下98%，4w以上95%"
   - 根据标准售价自动选择折扣率

4. **减价优惠**
   - 如"最大优惠-2000"、"标准售价最大优惠-1500"
   - 计算公式：标准售价 - 优惠金额

5. **适用优惠政策**
   - 如果产品有适用优惠政策，使用政策中的计算方式

## 查询方式

- **全部**：在所有字段中搜索
- **产品名**：仅在产品名称中搜索
- **交付方**：在交付部门中搜索
- **行业**：在行业字段中搜索
- **地点**：在地点字段中搜索
- **公司名**：在公司名称中搜索

## 数据更新

### 更新产品数据

1. 更新 `products_with_policies.json` 文件
2. 确保 JSON 格式正确
3. 推送到 GitHub（如果使用 Vercel）
4. 系统会自动加载最新数据

### 数据格式

产品数据应包含以下字段：

```json
{
  "产品名": "产品名称",
  "交付部门": "交付部门",
  "标准售价": "价格",
  "事业部最大优惠": "优惠信息",
  "最大优惠后价格": "优惠后价格",
  "来源Sheet": "来源",
  "来源文件": "来源文件",
  "搜索关键词": "关键词",
  "是否实习类": false,
  "适用优惠政策": {
    "优惠": "优惠描述",
    "赠课": "赠课信息",
    "适用范围": "适用范围",
    "计算方式": "计算方式"
  }
}
```
